# Pomodoro Timer

A beautiful, feature-rich Pomodoro timer to boost your productivity and track your focus sessions.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

### Core Timer
- Customizable focus duration (default 25 minutes)
- Short break (default 5 minutes)
- Long break after set number of sessions (default 15 minutes)
- Quick time adjustment buttons (+1m, +5m, -1m, -5m)
- Pause, resume, skip, and reset controls
- Visual progress ring with countdown display
- Session counter with daily goal tracking

### Task Management
- Add tasks with estimated number of sessions
- Categories: Work, Study, Personal, Other
- Track completed sessions per task
- Set active task for current focus session
- Mark tasks as complete

### Statistics
- Total focus time tracked
- Total sessions completed
- Daily streak counter with best streak
- Weekly productivity chart
- Task breakdown by category

### Customization
- 6 color themes: Midnight, Forest, Ocean, Sunset, Matcha, Lavender
- Background ambient sounds: Rain, Forest, Cafe, Lo-fi
- Selectable alert sounds: Bell, Chime, Digital
- Adjustable volume for ambient sounds
- Auto-start breaks and focus sessions (toggleable)
- Professional browser notifications

### Data Management
- Export all data as JSON
- Import data from backup
- Clear all data option
- All data stored locally in your browser

### Mobile
- Fully responsive design
- PWA support (install on home screen)
- Touch-friendly controls
- Keyboard shortcuts for desktop

## Keyboard Shortcuts

| Key | Action |
|:---:|:------:|
| Space | Start / Pause timer |
| Right Arrow | Skip current session |
| Left Arrow | Reset timer |
| 1 | Switch to Focus mode |
| 2 | Switch to Short Break |
| 3 | Switch to Long Break |

## How to Use

1. Open the app in your browser
2. Add tasks you want to work on (optional)
3. Select a background sound if desired
4. Click the Play button to start a focus session
5. Work until the timer ends
6. Take a break when prompted
7. Repeat to build your streak
